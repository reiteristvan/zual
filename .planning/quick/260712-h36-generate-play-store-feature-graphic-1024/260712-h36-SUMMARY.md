# Quick Task 260712-h36: Play Store Feature Graphic Summary

**Generated store_assets/feature_graphic.png (1024x500) by headlessly re-rendering the real SunrisePainter at progress 0.85, with a non-auto-discovered generator and a companion read-only drift-lock test.**

## Performance

- **Duration:** 12 min
- **Tasks:** 2
- **Files modified:** 3 (2 created source files, 1 generated binary)

## Accomplishments
- `store_assets/feature_graphic.png` — a valid 1024x500 RGBA PNG composited directly from `SunrisePainter(progress: 0.85, twinklePhase: 0.0)`, depicting the sun above a warm green hill with stars/moon already faded out; no text, logo, or device frame.
- `test/tool/generate_feature_graphic.dart` — the explicit generator, deliberately not matching `*_test.dart` so a bare `flutter test` never rediscovers/overwrites it; documents the `dart run` -> `flutter test` deviation and reasoning in its header doc comment.
- `test/tool/feature_graphic_test.dart` — a read-only regression test that re-renders in-memory using the same shared `kHeroProgress`/`kFeatureGraphicSize` consts (imported, not restated) and byte-diffs against the committed PNG, failing loudly with an actionable `reason:` message on drift.
- Verified end-to-end: full `flutter test` suite (134 tests) passes, and `store_assets/feature_graphic.png`'s md5 hash is unchanged before and after the full run, confirming the WR-04 anti-pattern (silent overwrite on every test run) is not repeated.

## Files Created/Modified
- `test/tool/generate_feature_graphic.dart` - Explicit (non-`_test.dart`) generator; reuses `SunrisePainter` + `renderPainterToPng`; exports `kFeatureGraphicSize`/`kHeroProgress` consts; writes and self-verifies `store_assets/feature_graphic.png`.
- `test/tool/feature_graphic_test.dart` - Read-only `_test.dart` regression test; re-renders in-memory with the imported shared consts and byte-diffs against the committed PNG; also asserts committed-file dimensions.
- `store_assets/feature_graphic.png` - The generated 1024x500 RGBA PNG feature graphic (binary asset).

## Decisions Made
- Reused `SunrisePainter`/`renderPainterToPng` verbatim rather than re-deriving any paint code, per the plan's explicit must-have and key_links.
- `progress = 0.85` selected as the hero moment (see key-decisions in frontmatter for the star/moon-opacity clamp math backing this choice).
- Shared generation-parameter consts live in the generator file and are imported (not copy-pasted) by the regression test, per the plan's Task 2 guidance, to guarantee the two files cannot drift independently.
