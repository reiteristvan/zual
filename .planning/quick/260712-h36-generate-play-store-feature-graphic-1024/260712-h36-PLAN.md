<objective>
Generate the Play Store feature graphic — a 1024x500 PNG banner — for Zual, code-composited
from the real Night to Sunrise scene painter and headless-rendered through the existing Skia
render-to-PNG utility, exactly as the existing 512x512 store icon is produced.

Purpose: The Play Console listing requires a 1024x500 feature graphic. It must be genuinely
pixel-consistent with the shipping app's Night to Sunrise scene (reuse the real painter), and it
must NOT repeat the icon generator's flagged anti-pattern (05-REVIEW.md WR-04/WR-05) of silently
overwriting a committed binary on every `flutter test` run.

Output: store_assets/feature_graphic.png (1024x500 RGBA PNG), an explicit generation tool, and a
read-only drift-detecting regression test.
</objective>

    purpose: a plain `flutter test` discovers only `test/**/*_test.dart`, so this file is never

    in test/tool/icon_renderer.dart — so it is invoked explicitly via
    `flutter test test/tool/generate_feature_graphic.dart`, not `dart run`. State this deviation
    from the literal `dart run` suggestion, and its reason, in the file's header doc comment.

    Structure it like test/tool/generate_store_icon_test.dart: a single `testWidgets(...)` whose
    body runs entirely inside `tester.runAsync(() async { ... })`.

    Reuse the real in-app painter and the existing renderer — do NOT re-derive any paint code or
    colors:
      - import 'package:zual/scenes/sunrise/sunrise_painter.dart' (SunrisePainter) — confirm the
        package name from pubspec.yaml `name:` and use that in the import.
      - import the renderPainterToPng helper from 'icon_renderer.dart' (same directory).

    Define two named consts with rationale comments and render at a representative "hero" moment:
      - kFeatureGraphicSize = Size(1024, 500) — the Play Store feature-graphic dimensions.
      - kHeroProgress = 0.85 — a late/near-done state: warm sunrise, sun visible above a warm
        green hill, stars and moon faded out (starOpacity/moonOpacity clamp to 0 well before this
        progress). This reads best as a banner. Pass twinklePhase: 0.0 (irrelevant since no stars
        render at this progress).

    Construct SunrisePainter(progress: kHeroProgress, twinklePhase: 0.0), call

    ui.instantiateImageCodec yields width 1024 and height 500. Dispose the decoded frame image.

    Do NOT add any text, caption, wordmark, logo, or device frame — SunrisePainter draws only the
    sky gradient, stars, moon, sun, and hill, which is exactly what is wanted.

    After writing the file, run the generator to produce the committed asset (see verify).
