<objective>
Turn timer completion into the calm finished state (CTRL-03, CTRL-04). On the edge into
`TimerPhase.done`, play the two-tone chime once (unless muted) through the injected ChimePlayer,
remove the old auto-pop so the settled scene stays on screen, and reveal a gently breathing
"All done — tap when ready" pill that returns to Setup on tap. The scene's own progress==1 end
visual is already correct from Phase 3 — this plan layers the chime and pill on top of it.

**Phase user story (MVP):** As a parent, I want a finished timer to settle into a calm, wordless
end state with a single soft chime and a clear "all done" cue, so that my child is gently
signaled that time is up without an alarm or celebration. (Derived from the ROADMAP Phase 4 goal.)

Purpose: Complete the phase's user-facing slice — the dwelled-in `done` state the app never
had before — replacing Phase 3's auto-pop scaffolding.
Output: Done-edge chime trigger + breathing pill in `RunningScreen` + completion widget tests.
</objective>
