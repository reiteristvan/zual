## Phase Goal (MVP user story)

**As a** parent, **I want to** configure a countdown's duration and scene and launch it with my last-used settings pre-selected, **so that** my child can watch a wordless visual timer without me re-entering settings each time.

This plan is the production-polish slice (MVP task-order step 5): the feature already works end-to-end; here it is made pixel-accurate.
