Zual — Privacy Policy

Effective date: 2026-07-09

Zual is a visual countdown timer for young children. This page explains, plainly and completely, how the app handles data. There isn't much to explain, because Zual collects nothing.

No accounts, no data collection

Zual has no user accounts and no login of any kind. The app does not collect, store, or transmit any personal data. It does not use analytics, does not include advertising, and does not contain any third-party tracking software development kits (SDKs).

Works fully offline

Zual works entirely offline. It never sends anything over the network, because it has no server and no internet-connected features.

What is stored on the device

The only information Zual keeps is the last duration and visual scene a parent chose, so the app can pre-select them the next time it's opened. This is stored locally on the device using the operating system's standard app storage. It is never sent anywhere, and it never leaves the device.

Children's privacy

Zual is parent-operated: a parent sets the duration and picks a scene, and the child only watches the resulting visual — there is nothing on the running screen for a child to tap or interact with. Zual collects no information from anyone, including children. There is no chat, no user-generated content, and no way for a child to communicate with anyone through the app.

No ads, no third parties

Zual contains no advertising and shares no data with any third party, because it collects no data to share in the first place.
